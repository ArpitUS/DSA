# DSA Practice in Go & Python

A focused repository for practicing **Data Structures & Algorithms (DSA)** using **Python** and **Go**. The goal is to build interview-ready problem solving skills for backend, SRE, cloud, and software engineering roles.

---

## 🚀 Objectives

* Master core DSA patterns
* Solve problems quickly in Python
* Re-implement clean production-style solutions in Go
* Improve time complexity analysis
* Prepare for coding interviews

---

## 📚 Languages Used

* **Python** → Fast prototyping, interview speed
* **Go** → Strong typing, backend-oriented implementation

## 📅 Daily Routine

### Python (45–60 min)

* Solve 1–2 timed problems
* Focus on speed and optimization

### Go (45–60 min)

* Rebuild one problem from Python
* Focus on readability and structure

### Review (15 min)

* Time complexity
  n- Edge cases
* Alternative approaches
